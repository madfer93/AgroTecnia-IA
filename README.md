# AgroTecnia JyM B2B
Landing Page corporativa diseñada específicamente para el sector rural y agropecuario (SaaS Agrícola), desarrollada por **JyM Tech Solutions (RUES 495502)**.

## Características Principales
* **Desarrollo:** React + Vite
* **Estilos:** Tailwind CSS
* **Iconografía:** Lucide React
* **Despliegue Objetivo:** Vercel

## Estructura de Módulos (Mobile First)
1. **Hero:** Integración de Inteligencia Artificial para el agro colombiano.
2. **Servicios:** Agronomía (Bitácoras WhatsApp), Zootecnia (Vacunación), Visión IA (Plagas), Clima Satelital.
3. **Formulario de Piloto/Diagnóstico:** Capta leads directamente (nombre, ubicación, tipo de producción, dolores del campo).
4. **Trust Signals:** Footer institucional con verificación de cámara de comercio y políticas de Habeas Data (Ley 1581).

## Instalación Local
```bash
npm install
npm run dev
```

Este proyecto no requiere backend externo inicial para captar atención (Make), ya que la arquitectura de Supabase u otras bases de datos se podrá vincular directamente en `App.jsx`.
